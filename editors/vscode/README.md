# portlang

VSCode extension for [portlang](https://github.com/portofcontext/portlang) `.field` files.

## portlang installation
```shell
brew tap portofcontext/homebrew-tap
brew install portlang
```

## Features

- Syntax highlighting for `.field` files

## Requirements
- portlang .field files

## Usage

Open any `.field` file — the language server starts automatically.
